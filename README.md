# Birthday Greeting Page

This project is a **Birthday Greeting Page** ("Página de Felicitaciones") designed as a personalized web card to celebrate someone's special day. The page features a combination of warm wishes, a personal video message, and memorable photos, all wrapped in a visually appealing and responsive design.

## Features

- **Personalized Messages:** Display heartfelt birthday wishes with customizable text.
- **Video Section:** Embed a video message for a more personal touch.
- **Photo Gallery:** Show a collection of memorable images.
- **Beautiful Design:** Uses appealing fonts and colors for a festive look.
- **Responsive Layout:** Optimized for both desktop and mobile devices.

## Technologies Used

- **HTML5:** Markup for the web page structure.
- **CSS3:** Styling with custom fonts from Google Fonts (`Dancing Script`, `Delius`), gradients, and responsive design.
- **(Optional) JavaScript:** *Not included by default, but can be added for interactivity.*

## Project Structure

## How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Lednny/Pagina_De_Felicitaciones.git
   ```
2. **Add your own images** in the `images/` folder.
3. **Edit `index.html`** to personalize the message, the recipient’s name, and the sender’s name.
4. **Add your video:** Place your `.mp4` video in a suitable location and update the `<source src="">` in the video section.
5. **Open `index.html` in your browser** to view and share your customized birthday page!

## Customization

- Change colors, fonts, or layout in `src/Tarjeta.css`.
- Replace or add more images in the `images/` folder.
- Update the texts in `index.html` for a unique and personal greeting.
